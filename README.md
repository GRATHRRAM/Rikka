# Rikka
Simple OpenGL Engine

# How to Compile
```bash
# Clone Repo
cd Rikka
mkdir build
cd build
cmake ..
make 
```
# Arch Dependencies 
```
sudo pacman -Sy glew glfw make cmake gcc
```
